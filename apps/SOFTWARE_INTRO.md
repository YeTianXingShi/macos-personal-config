# 常用开发工具介绍

本文档介绍了 `apps/Brewfile` 中列出的各类软件工具，涵盖了命令行工具、桌面应用及字体配置，旨在提升 macOS 环境下的开发效率和系统体验。

## 命令行工具 (CLI Tools)

这些工具主要在终端中使用，能够显著增强你的日常开发工作流。

*   **bat**: 
    `cat` 命令的加强版。它支持多种编程语言的语法高亮，并集成了 Git 状态显示（在侧边栏显示修改行），同时支持分页查看长文件。
*   **eza**: 
    `ls` 命令的现代替代品。它是原 `exa` 项目的活跃维护分支，支持彩色输出、文件图标显示、更友好的列格式以及内置的目录树视图。
*   **fzf**: 
    极其强大的命令行模糊查找器。它可以与各种命令集成，让你通过模糊搜索快速定位文件、历史记录、进程、主机名等，是提升终端操作速度的神器。
*   **gh**: 
    GitHub 官方命令行工具。它将 GitHub 的工作流（如管理 Issue、Pull Request、Release 等）带到了终端，让你无需离开命令行即可完成大部分协作任务。
*   **micro**: 
    一个直观、现代的终端文本编辑器。它支持鼠标操作和通用的快捷键（如 Ctrl+S 保存，Ctrl+C 复制），上手零门槛，是 Vim/Emacs 之外的轻量级替代方案。
*   **mole**: 
    由 `tw93` 开发的 **Mac 深度清理与优化工具**。它能够帮助用户深度清理系统缓存、移除无用的残留文件，从而释放磁盘空间并优化系统运行速度。它是保持 macOS 干净清爽的实用小工具。
*   **node@24**: 
    Node.js 是基于 Chrome V8 引擎的 JavaScript 运行时。版本 24 代表了较新的特性集，是进行现代前端开发和后端服务构建的核心基础。
*   **opencode**:
    AI 辅助编程工具的核心组件，支持与多种大语言模型交互，辅助代码编写、调试和重构。
*   **zoxide**: 
    一个更智能的 `cd` 命令（基于 Rust 编写）。它会记录你访问过的目录，之后你只需输入 `z` 加上目录名的一部分，它就能通过权重算法自动跳转到目标位置。

## 桌面应用 (GUI Apps/Casks)

这些是带有图形界面的应用程序，通过 Homebrew Cask 进行管理。

*   **Ghostty**: 
    一款备受期待的、高性能的跨平台终端模拟器。它使用 Zig 语言编写，利用 GPU 加速渲染，旨在提供极低延迟的操作体验和现代化的界面配置。
*   **Google Chrome**:
    Google 开发的网页浏览器，以其快速的性能、强大的开发者工具和丰富的扩展生态而闻名，是前端开发的标准测试环境。
*   **Hoppscotch**: 
    一个轻量级、开源的 API 开发测试平台。它是 Postman 的优秀替代品，支持 REST、GraphQL、WebSocket 等协议，能极大地简化接口调试流程。
*   **PixPin**:
    一款功能丰富的截图与贴图工具。它支持长截图、屏幕录制、OCR 文字识别以及将截图“贴”在屏幕上作为参考，极大提升了多任务处理效率。
*   **Stats**:
    一款开源的 macOS 菜单栏系统监控工具。它可以实时显示 CPU、内存、磁盘、网络和电池等硬件状态，界面简洁且高度可定制。
*   **Telegram Desktop**:
    Telegram 的官方桌面客户端。这是一款注重速度和安全性的即时通讯软件，支持跨平台同步和强大的群组功能。
*   **Typora**:
    一款所见即所得（WYSIWYG）的 Markdown 编辑器。它去除了预览窗口与源码窗口的割裂感，提供极致简洁的写作体验。
*   **Visual Studio Code**:
    微软开发的免费、开源代码编辑器。凭借其轻量级、强大的插件生态和优秀的智能提示功能，它已成为全球最流行的 IDE 之一。
*   **Xmind**: 
    全球领先的思想导图和头脑风暴软件。无论是项目规划、逻辑梳理还是知识库构建，Xmind 都能通过直观的可视化方式帮助你理清思路。

## VS Code 扩展 (Extensions)

为了提升 `Visual Studio Code` 的开发体验，Brewfile 中还预设了以下扩展：

*   **AI 辅助**: `GitHub Copilot` & `Chat`, `Google Gemini Code Assist` —— 智能代码补全与对话助手。
*   **Python 开发**: `Python`, `Pylance`, `Debugpy` —— 提供 Python 语言支持、智能提示和调试功能。
*   **界面美化**: `vscode-icons` —— 为文件资源管理器提供丰富的文件图标；`Chinese Language Pack` —— 中文语言包。

## 字体 (Fonts)

*   **font-maple-mono-nf-cn**: 
    **Maple Mono (枫叶等宽字体)**。这是一款开源的圆体风格等宽字体，专为代码编写设计。`nf` 表示集成了 Nerd Fonts 图标，`cn` 表示针对中文显示进行了优化，非常适合作为终端或 IDE 的主力字体。
