# 常用开发工具介绍

本文档介绍了 `apps/Brewfile` 中列出的各类软件工具，涵盖了命令行工具、桌面应用及字体配置，旨在提升 macOS 环境下的开发效率和系统体验。

## 命令行工具 (CLI Tools)

这些工具主要在终端中使用，能够显著增强你的日常开发工作流。

*   **bat**: 
    `cat` 命令的加强版。它支持多种编程语言的语法高亮，并集成了 Git 状态显示（在侧边栏显示修改行），同时支持分页查看长文件。
*   **eza**: 
    `ls` 命令的现代替代品。它是原 `exa` 项目的活跃维护分支，支持彩色输出、文件图标显示、更友好的列格式以及内置的目录树视图。
*   **fzf**: 
    极其强大的命令行模糊查找器。它可以与各种命令集成，让你通过模糊搜索快速定位文件、历史记录、进程、主机名等，是提升终端操作速度的神器。
*   **zoxide**: 
    一个更智能的 `cd` 命令（基于 Rust 编写）。它会记录你访问过的目录，之后你只需输入 `z` 加上目录名的一部分，它就能通过权重算法自动跳转到目标位置。
*   **mole**: 
    由 `tw93` 开发的 **Mac 深度清理与优化工具**。它能够帮助用户深度清理系统缓存、移除无用的残留文件，从而释放磁盘空间并优化系统运行速度。它是保持 macOS 干净清爽的实用小工具。
*   **node@24**: 
    Node.js 是基于 Chrome V8 引擎的 JavaScript 运行时。版本 24 代表了较新的特性集，是进行现代前端开发和后端服务构建的核心基础。

## 桌面应用 (GUI Apps/Casks)

这些是带有图形界面的应用程序，通过 Homebrew Cask 进行管理。

*   **Ghostty**: 
    一款备受期待的、高性能的跨平台终端模拟器。它使用 Zig 语言编写，利用 GPU 加速渲染，旨在提供极低延迟的操作体验和现代化的界面配置。
*   **Hoppscotch**: 
    一个轻量级、开源的 API 开发测试平台。它是 Postman 的优秀替代品，支持 REST、GraphQL、WebSocket 等协议，能极大地简化接口调试流程。
*   **Xmind**: 
    全球领先的思想导图和头脑风暴软件。无论是项目规划、逻辑梳理还是知识库构建，Xmind 都能通过直观的可视化方式帮助你理清思路。

## 字体 (Fonts)

*   **font-maple-mono-nf-cn**: 
    **Maple Mono (枫叶等宽字体)**。这是一款开源的圆体风格等宽字体，专为代码编写设计。`nf` 表示集成了 Nerd Fonts 图标，`cn` 表示针对中文显示进行了优化，非常适合作为终端或 IDE 的主力字体。
